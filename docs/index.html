Mre_md5

App demonstrate how to get md5 for MRE platform mobile phone (including Nokia S30+). LEFT_SOFTKEY - select file, RIGHT_SOFTKEY - exit app, KEY_NUM1 - input text. Tested on Nokia 225 with display resolution 240x320. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application files - "mre_md5.vxp";
